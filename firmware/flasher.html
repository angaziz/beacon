BEACON // web flasher

Install the Beacon firmware on a Waveshare ESP32-S3-Touch-AMOLED-2.16 straight from the browser. No toolchain, no drivers on macOS.

  1. Connect the device with a USB-C data cable.
  2. Use Chrome or Edge on desktop (Web Serial is not available in Safari/Firefox).
  3. Click Install and pick the device's serial port — it shows up with a generic name like usbmodemXXXX or ESP32-S3. No drivers needed: the board uses the ESP32-S3's native USB (ignore the CP2102/CH34x driver links in the "no port" help dialog).
  4. First install? Choose the "erase device" option when prompted.

Port missing from the picker? Close anything using the serial port (serial monitors, Arduino IDE), or force the ROM bootloader: unplug, hold BOOT, plug in while holding for ~3 s, release, then try again.

After flashing, the device opens a Beacon-setup WiFi hotspot — join it and the captive portal asks for your network.

Beacon is a personal, unofficial project — source on GitHub. Flashing uses ESP Web Tools. Use at your own risk.
